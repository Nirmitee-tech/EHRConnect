## Color Scheme

### Status Colors
- **Active**: Green (#10B981)
- **Stopped**: Red (#EF4444)
- **Completed**: Gray (#6B7280)
- **Draft**: Blue (#3B82F6)

### Mode Colors
- **Quick Add**: Green (#10B981) - Fast, efficient
- **Standard**: Blue (#3B82F6) - Professional, complete
- **Template**: Purple (#8B5CF6) - Smart, automated
- **Edit**: Blue (#3B82F6) - Modification mode

### Instruction Highlight
- Background: Amber-50 (#FFFBEB)
- Border: Amber-200 (#FDE68A)
- Text: Amber-700 (#B45309)

## Responsive Design

### Desktop (>1024px)
- Full layout with all features
- Side-by-side template grid (2 columns)
- Expanded form fields

### Tablet (768px - 1024px)
- Compact layout
- Single column templates
- Reduced spacing

### Mobile (<768px)
- Stack all elements vertically
- Full-width buttons
- Touch-friendly targets (44px min)
- Collapsible sections

## Accessibility Features

### Keyboard Navigation
- Tab through all interactive elements
- Enter to activate buttons
- Escape to close dialogs
- Arrow keys for dropdowns

### Screen Reader Support
- ARIA labels on all controls
- Role attributes
- Live regions for notifications
- Semantic HTML

### Visual Indicators
- Focus rings on active elements
- High contrast mode support
- No color-only information
- Clear error messages
